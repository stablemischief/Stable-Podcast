# Stable Podcast

A standalone, AI-powered podcast generation tool. Create professional podcasts from any content — with full control over speakers, episode templates, transcript editing, and multi-provider TTS synthesis.

## Status

**In Planning** — Architecture and tech stack are being defined. See the `planning/` directory for the extraction analysis from [Open Notebook](https://github.com/lfnovo/open-notebook).

## Features (Planned)

- **AI Script Generation** — Generate podcast outlines and transcripts from source content using configurable AI providers
- **Speaker Profiles** — Define speakers with unique voices, personalities, and TTS voice IDs (1-4 speakers per episode)
- **Episode Templates** — Reusable podcast format templates with configurable AI models, segment counts, and briefings
- **Transcript Editor** — In-browser transcript review and editing before audio generation
- **Multi-Provider TTS** — Text-to-speech via OpenAI, ElevenLabs, Google Cloud, or local providers
- **Audio Mixing** — FFmpeg-powered audio mixing and mastering
- **Simple Mode** — Focused daily podcast studio: upload PDF, review transcript, generate audio

## Origin

Extracted from [stablemischief/Stable-Notebook](https://github.com/stablemischief/Stable-Notebook), which is a fork of [lfnovo/open-notebook](https://github.com/lfnovo/open-notebook). The goal is a lightweight, purpose-built podcast tool without the full notebook/research platform overhead.
